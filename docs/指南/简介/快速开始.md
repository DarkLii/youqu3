# 快速开始

## 安装

基础环境（默认支持命令行自动化测试）：

```
pip install youqu3
```

![](/install.gif)

桌面 UI 自动化测试：

```
pip install youqu3[desktop-ui]
```

Web UI 自动化测试：

```
pip install youqu3[webui]
```

D-Bus 接口自动化测试：

```
pip install youqu3[dbus]
```

HTTP 接口自动化测试：

```
pip install youqu3[http]
```

远程执行：

```shell
pip install youqu3[remote]
```

## 创建用例工程

```shell
mkdir my_autotest
cd my_autotest/
youqu3 init
```

![](/init.gif)
